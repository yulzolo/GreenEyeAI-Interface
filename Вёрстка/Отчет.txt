Отчёт о проделанной работе
Тема: Разработка веб‑интерфейса системы мониторинга умной теплицы.

1. Цель работы. 
Выполнить вёрстку многостраничного (в рамках одной HTML‑страницы) веб‑интерфейса на основе макета, предоставленного в Figma. Результат должен представлять собой адаптивный, корректный и визуально соответствующий макету сайт с функциональным боковым меню и прокручиваемыми блоками.

2. Выполненные задачи. 
Проанализирован макет Figma: определены основные блоки, размеры, отступы, цветовая гамма, шрифт. Создана HTML‑структура, включающая боковое меню для переключения основных разделов (дашборд, полки, вода, свет, заболевания) и отдельные блоки. Разработана CSS‑стилизация. Реализовано переключение разделов. Добавлены интерактивные элементы: переключатели подачи воды и распрыскивания с визуальной обратной связью. Учтены точные размеры блоков согласно макету. Обеспечена семантическая вёрстка и использование шрифта Inter.

3. 
Изученные технологии и освоенные навыки. HTML5: построение семантической структуры, использование тегов section, aside, nav. CSS3: применение Flexbox для выравнивания элементов, Grid для позиционирования, анимации (плавное появление блоков). Реализована обработка событий клика, динамическое изменение классов, переключение видимости разделов, управление состоянием переключателей. Работа с макетом Figma: извлечение размеров, цветов, отступов. Тестирование в современных браузерах (Chrome, Firefox, Edge), обеспечение корректного отображения на экранах шириной от 320px до 1440px+.

4. 
Использованные ресурсы. Документация: MDN Web Docs (HTML, CSS, JavaScript), CSS‑Tricks (руководства по Flexbox и Grid). Сообщества: Stack Overflow (решение конкретных проблем позиционирования и скриптов). Инструменты: Figma (анализ макета), VS Code (среда разработки),  YouTube, DeepSeek, Qwen. Дополнительные материалы: статьи по адаптивному дизайну.

5. 
Результаты. Разработан полностью функционирующий статический сайт, визуально идентичный исходному макету. Все элементы интерфейса интерактивны: меню переключает разделы, тумблеры меняют состояние. Код структурирован, готов к дальнейшей интеграции с реальными данными (подключение датчиков, камер и т.п.).

6. 
Выводы. В ходе работы успешно освоены базовые принципы современной веб‑вёрстки, получен опыт перевода графического макета в рабочий HTML/CSS-код. Сформированные навыки позволяют в дальнейшем разрабатывать полноценные клиентские интерфейсы любой сложности.
